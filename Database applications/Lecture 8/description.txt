 Book - книги. Хранит название, категорию, автора, год выпуска, цену и статус (свободна, зарезервирована, выдана, выдана и зарезервирована).
 Client - посетители. Хранит ФИО, адрес, номер телефона, эл. почту.
 Employee - библиотекари. Хранит те же данные что и Clients.
 Requsets - заявки. Хранят ID клиента, ID книги и статус (выполнена, отменена, ожидает выполнения).
 Issue - выданные книги. Хранит ID книги, дату выдачи, планируемую дату возврата, ID заявки, ID клиента и ID библиотекаря.
 Return - принятые книги. Хранит ID выдачи, фактическую дату возврата, ID клиента и ID библиотекаря.
